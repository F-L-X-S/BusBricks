\doxysubsection{Detailed Description}
Represents a message with information for sender and receiver, and provides methods for converting between PDU and message content. 

This class handles the mapping between string-\/formatted PDUs (Representation) and text messages with sender-\/ and receiver information (\doxylink{class_content}{Content}). It offers constructors for creating {\ttfamily \doxylink{class_message}{Message}} objects from different types of inputs and provides methods to convert between a message\textquotesingle{}s content and its PDU representation.

\begin{DoxyNote}{Note}
The {\ttfamily \doxylink{class_message}{Message}} class inherits from {\ttfamily \doxylink{class_content}{Content}\texorpdfstring{$<$}{<}\doxylink{struct_message__content__t}{Message\+\_\+content\+\_\+t}, String\texorpdfstring{$>$}{>}}, providing a specialized implementation for handling messages.
\end{DoxyNote}
