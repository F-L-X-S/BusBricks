\doxysubsection{Detailed Description}
Mock class to simulate the \doxylink{classarduino_mocking_1_1_software_serial}{Software\+Serial} library for native builds. 

This class is a mock implementation of the Arduino {\ttfamily \doxylink{classarduino_mocking_1_1_software_serial}{Software\+Serial}} class, designed for testing and simulation in non-\/\+Arduino environments. It provides methods to simulate serial communication including reading, writing, and flushing data.\hypertarget{classarduino_mocking_1_1_software_serial_Public}{}\doxysubsection{\texorpdfstring{Methods}{Methods}}\label{classarduino_mocking_1_1_software_serial_Public}
